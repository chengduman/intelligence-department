# 📡 情报部 · Intelligence Department

**部长：马俊伟** | 下属：3人（黄思远、罗文杰、白若溪）

## 部门定位
公司的"雷达"和"哨兵"，负责行业动态监控、竞品追踪、政策法规监测、风险预警和每日情报推送。

## 工作模式

| 模式 | 说明 | 周期 |
|------|------|:----:|
| 日常监控 | 设定关键词和信源，持续跟踪 | 每日 |
| 定时推送 | 每日/每周情报简报 | 按设定 |
| 专题深挖 | 对特定事件/企业深度挖掘 | 按需 |
| 紧急预警 | 重大突发事件即时预警 | 即时 |

## 本仓库用途
- 📰 每日情报简报归档
- 🎯 竞品动态追踪与分析
- 📋 政策法规监测报告
- ⚠️ 风险预警记录

## 分派任务流程
1. CEO确定重点关注方向
2. 马俊伟部长分配监控任务（分析师各负责不同领域）
3. 情报分析师提交情报简报/预警
4. 部长审核情报质量 → 推送至相关人员
